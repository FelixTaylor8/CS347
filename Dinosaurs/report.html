Project Report

  1. This site's audience is a general one of "people who like dinosaurs," and they can learn about them or find dinosaur movies/games.
  2. The info I used was written in my own words or quoted and sourced, and the images were all public domain/creative commons. The time period page uses lorem ipsum text because it was taking way too long to do the research for the other pages.
  3. The home page contains a form and information about what each of the pages is for. The other four are all various dino
  4. The header has the T-rex picture and the title
  5. Navigation links are at the top
  6. The t-rex disappears on a small screen, the multi-column pages turn to one column
  7. Each page uses the same header and the same gray and blue sections
  8. There are images of dinosaurs and fossils with alt tags describing them
  9. The favicon of each page is a little dinosaur cartoon
  10. Images and CSS are in their own folders, with the exception of splash.jpg, which had to go in the CSS folder as for some reason it couldn't be reached otherwise, even when I used the direct path
  11. Have not imported any libraries or frameworks
  12. Everything validates when I run it
  13. Most text is aligned to the left with the exception of some headers and links
  14. Content is confined to padded blue and gray sections
  15. Headers and nav links use Verdana, everything else uses Trebuchet
  16. The home page has a form for signing up to an imaginary newsletter
  17. There are links to YouTube videos, image sources, and information sources
  18. Code uses kebab-case and was run through auto formatters
  19. Images (besides the big t-rex) fade in and links transition to a different shade when hovered
  20. Stored on GitHub
  21. File names correct
  22. Works with https
  23. Uses port 443
  24. Redirects to https
  25. Pushed when I remembered to
  26. Report here
