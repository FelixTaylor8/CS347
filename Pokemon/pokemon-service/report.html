1. This service allows users to input nickname ideas and fun facts about specific Pokemon (works with the first 99 Pokemon).
  2. Used MySql to store valid Pokemon, nicknames, and fun facts.
  3. Used 
    const express = require('express');
const service = express();
service.use(express.json());
    and service.get/post/patch/delete interacts with the pokemon database
  4. Works at https://project2.apir.me:8443
  5. Names used: Pokemon:
    • service.get('/pokemon' - all pokemon
    • service.get('/pokemon/:mon' - specific pokemon by name
    • service.get('/pokemon/id/:id' - specific pokemon by id
    • service.patch('/pokemon/:name/like' - like a pokemon by name
    • service.patch('/pokemon/id/:id/like' - like a pokemon by id
    Nicknames:
    • service.get('/nick' - all nicknames
    • service.get('/nick/:mon' - specific pokemon's nicknames (use pokemon's name)
    • service.get('/nick/id/:id' - specific nickname
    • service.post('/nick' - post a nickname (requires body with elements "id", (doesn't auto-increment) "nick," and "mon")
    • service.patch('/nick/:id/like' - like a specific nickname
    • service.patch('/nick/:id/report' - report a specific nickname
    • service.patch('/nick/:id/removereport' - remove report from nickname
    • service.patch('/nick/:id/approve' - approve a nickname
    • service.delete('/nick/:id' - delete a nickname
    Facts:
    • service.get('/fact' - all facts
    • service.get('/fact/:mon' - specific pokemon's facts
    • service.get('/fact/id/:id' - specific fact
    • service.post('/fact' - post a fact (requires body with elements "id", (doesn't auto-increment) "fact," and "mon")
    • service.patch('/fact/:id/like' - like a specific fact
    • service.patch('/fact/:id/report' - report a specific fact
    • service.patch('/fact/:id/removereport' - remove report from fact
    • service.patch('/fact/:id/approve' - approve a fact
    • service.delete('/fact/:id' - delete a fact
  6. service functions use response.json
  7. Able to get pokemon/nicknames/facts, post new nicknames/facts, report/approve and like nicknames/facts, and delete nicknames/facts
  8. Database accessible in the droplet. ufw blocks other ports
