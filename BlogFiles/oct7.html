I watched this video about DOM traversal, here are some notes:

  • Elements selected by ID can be passed as arguments to functions
  • Elements can be selected by class, and the resulting collection can be turned into an array with
    • Array.from(document.getElementByClassName(""));
  • querySelector does the same thing as getElementById, using CSS selectors instead of HTML IDs/classes
  • If there's more than one element with the same class, querySelector will use the first one
  • querySelectorAll gets all of the classes, and it can use forEach, unlike getElement
  • .children can select an element's children (ofc), which should also be converted to an array
  • .parentElement can select an element's parent, and parents can be skipped by using .closest
  • .nextElementSibling gets an element's next sibling, .previousElementSibling does the opposite

I also watched this one about event listeners

  • Bubbling refers to event listeners working on the closest element and moving up
  • Capturing refers to event listeners working on the furthest first
  • Event listeners can use capturing instead of bubbling by doing this:
    • element.addEventListener("click", e => { },
          {capture: true}
        )
  • .stopPropagation() prevents it from looking for other event listeners
  • { once: true } will prevent an event listener from being used again -- no global bools!
  • Can also use .removeEventListener, but it doesn't work for anonymous functions
